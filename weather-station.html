🌡️ Desktop Weather Station

A compact, offline weather monitoring device with custom PCB design, featuring real-time temperature and humidity display on a 0.96" OLED screen.

Project Demo

Project Overview

This desktop weather station is a fully custom design from the ground up. Built around an ATmega328PB microcontroller on a custom PCB, it continuously monitors temperature and humidity using a high-precision SHT31 sensor. The data is displayed in real-time on a compact 0.96-inch OLED screen, providing at-a-glance environmental monitoring for your workspace.

Designed with simplicity in mind, this weather station requires no network configuration or WiFi setup - it's completely offline and ready to use straight out of the box. Just plug in a USB-C cable for power, and you're done. The adjustable 3D-printed stand with rubber feet ensures stable placement on any desk surface.

Key Features

  • 🔌 Plug & Play No WiFi setup or configuration required - completely offline operation
  • 📊 Real-Time Monitoring Continuous temperature and humidity readings displayed on 0.96" OLED
  • ⚙️ Custom PCB Design Purpose-built circuit board with ATmega328PB microcontroller
  • 🔗 I2C Communication Efficient sensor communication protocol for reliable data
  • 💻 Arduino Compatible CH340C USB-to-serial chip for easy programming via Arduino IDE
  • 📐 3D Printed Stand Angle-adjustable mount with rubber feet for stability
  • 🔋 USB-C Power Modern power connection - works with any USB-C cable
  • 🎯 High Precision SHT31 sensor provides accurate environmental measurements

Design Philosophy

Simplicity First

No complicated setup or WiFi configuration. Just plug in and go - perfect for users who want environmental monitoring without the hassle of smart home integration.

Custom Hardware

Every aspect designed from scratch - from the PCB layout to the 3D-printed enclosure. This isn't a kit build; it's a fully custom engineered solution.

Offline Operation

Complete data privacy and reliability. No internet connection means no cloud dependencies, no data uploads, and no connectivity issues.

Maker-Friendly

Arduino-compatible programming interface makes it easy to customize the display, add features, or integrate additional sensors for your specific needs.
